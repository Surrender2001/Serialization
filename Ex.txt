Лабораторные работы по теме 9 и 10  Файловый и потоковый ввод и вывод. Сериализация.

1.Разработать класс для хранения данных о человеке (имя, фамилия, год рождения, адрес и т.п.). Реализовать методы сохранения данных о человеке в файл и загрузки данных о человеке из файла, используя механизм сериализации. Для демонстрации создать два приложения: одно для записи данных, другое - для чтения.


2.Разработать класс для хранения данных о человеке (имя, фамилия, год рождения, адрес и т.п.). Реализовать методы сохранения массива с людьми в текстовый файл и загрузки данных в массив из текстового файла. Текстовый файл должен быть удобного для чтения формата (например, в виде таблицы).
